2012 Code Happening

SCVSoft invites you to their

What is it?

A Code Happening is like a
Hackathon, but with a twist: in the
end, every participant's work gets
merged into something bigger
(and loosely artistic). The exact
final outcome is a surprise
for the participants

How can I
participate?

Just fork the repo, check out
the instructions on the readme and submit your work before Tuesday
Dec 18th EOD Pacific Time. You
can use any tech, just give
us instructions to run
your work.

Is there a prize?

Yes, there is! We are going to
evaluate each work according to
their idea, look and feel, code
style, etc. We are going to         
give a Kindle Paperwhite      
to the one we like
the best.

How can I
stay updated?

Follow us on twitter (@scvsoft).
We'll announce the winner,
and where to see the final
"happening" when it is
ready through that
channel.
